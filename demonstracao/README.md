# Demonstração de Uso — NotebookLM

Esta seção apresenta exemplos do funcionamento do NotebookLM desenvolvido neste projeto.

O objetivo da demonstração é ilustrar:
- como as fontes são utilizadas pelo NotebookLM;
- os tipos de perguntas realizadas;
- os formatos de resposta gerados.

Todas as respostas apresentadas são baseadas exclusivamente nas fontes carregadas no notebook.
