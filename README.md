# Conselheiro Financeiro Educacional — NotebookLM

Este projeto utiliza o **NotebookLM** como ferramenta de **Inteligência Artificial Generativa aplicada à educação financeira**.

O objetivo é organizar, sintetizar e explicar conceitos fundamentais relacionados à:
- saúde financeira
- planejamento mensal
- decisões financeiras de curto prazo

Todo o conteúdo gerado pelo notebook é **baseado exclusivamente em fontes teóricas**, sem risco de alucinação.

---

## Objetivo do Projeto
Criar um sistema de IA que funcione como um **guia educacional**, capaz de:
- explicar conceitos financeiros de forma clara
- organizar informações complexas
- gerar resumos, relatórios e materiais didáticos
- analisar situação financeira do usuário
- evitar gastos impulsivos

---

## Funcionalidades do Notebook
- Resumos conceituais
- Mapas mentais
- Relatórios explicativos
- Cartões didáticos
- Testes conceituais
- Infográficos
- Apresentações de slides

---

## Fontes
As respostas do NotebookLM são geradas **exclusivamente a partir das fontes carregadas**. Foram anexados:
- materiais de educação financeira
- artigos acadêmicos
- relatórios institucionais
- comando co contexto para melhor funcionamento da IA

---

## Tecnologias Utilizadas
- NotebookLM
- Inteligência Artificial Generativa
- Curadoria de fontes

---

## Observações
Este projeto tem finalidade **educacional e acadêmica**.
Não substitui aconselhamento financeiro profissional.
